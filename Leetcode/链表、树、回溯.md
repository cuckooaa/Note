### note（反转链表）
有 **递归（尾插法）** 和 **迭代（头插法）** 两种方式

*迭代（头插法）模板：*
```cpp
ListNode* reverseList(ListNode* head) {
    ListNode* pre = nullptr;
    ListNode* cur = head;
    while (cur) {
        ListNode* nxt = cur->next;
        cur->next = pre; // 把 cur 插在 pre 链表的前面（头插法）
        pre = cur;
        cur = nxt;
    }
    return pre;
}
```

## summary
- 删 / 插 / 改头 → 用 哨兵节点
- 遍历全部节点 → while(node != nullptr)
- 停在最后一个节点 → while(node->next != nullptr)
